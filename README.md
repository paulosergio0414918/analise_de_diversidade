Analisar os dados presentes e obter os índices 

[ ]Shannon-Wiener (H')
Índice de Shannon-Wiener (H'):
O que mede: A incerteza sobre a identidade de um indivíduo escolhido aleatoriamente. Quanto maior o valor, maior a diversidade e a complexidade da comunidade.
Como funciona: Baseado na teoria da informação, considera riqueza e equidade. Usa o logaritmo da proporção de indivíduos de cada espécie (p).

[ ] Simpson (C ou 1-C)
Índice de Simpson (C ou 1-C):
O que mede: A dominância de uma espécie (C) ou a probabilidade de dois indivíduos escolhidos serem de espécies diferentes (1-C). Quanto maior (1-C), maior a diversidade.
Como funciona: Calcula a probabilidade de dois indivíduos da amostra pertencerem à mesma espécie. Um valor mais baixo de C (ou mais alto de 1-C) indica maior diversidade.


[ ] Equitabilidade (J'):
O que mede: A uniformidade na abundância das espécies (quão parecidas são as populações de cada espécie). J' varia de 0 a 1.
Como funciona: Frequentemente usado junto com Shannon (H'/H'max), indica se a comunidade tem uma espécie dominante (J' baixo) ou se as espécies são bem distribuídas (J' próximo de 1).


[ ]Riqueza de Espécies (S):
Simplesmente o número total de espécies na área.

[ ]Abundância (Absoluta/Relativa/Densidade): 
Quantidade de indivíduos de cada espécie ou o total, essencial para calcular os índices.

[ ]Índices Beta (β):
Medem a diferença de espécies entre diferentes comunidades ou áreas (ex: β-1).
